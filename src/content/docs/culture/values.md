The gym floor rule applies to the codebase: everyone started somewhere, and nobody gets judged for working on their form.

## Judgement-free engineering

Questions in public channels, postmortems without blame, code review that critiques the code and never the person. Seniority means you make it easier for others to be wrong safely — not that you're right louder.

## Boring on purpose

We run proven tech at scale for millions of members paying $10/mo. That price only works because our ops are unexciting: managed services over self-hosted, one pattern over three clever ones, rollback over heroics.

## Write it down

Decisions get an ADR, questions get a handbook page, incidents get a postmortem. If it isn't written down, it didn't happen — and someone will have to figure it out again.
