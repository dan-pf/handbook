The gym floor rule applies to the codebase: everyone started somewhere, and nobody gets judged for working on their form. For the full set of tenets we operate by, see [our guiding tenets](/culture/tenets/).

## Judgement-free engineering

Questions in public channels, postmortems without blame, code review that critiques the code and never the person. Seniority means you make it easier for others to be wrong safely — not that you're right louder.

## Write it down

Decisions get an ADR, questions get a handbook page, incidents get a Correction of Errors. If it isn't written down, it didn't happen — and someone will have to figure it out again.
